# kotlin-multiplatform-example-with-typescript
Kotlin 1.2.* multi-platform experiments

Locations
=========
Shared: ./src
JS: ./calculator-js
Java: ./calculator-jvm

To build
========
Run in terminal:
    ./gradlew build

OR

    npm run build-kotlin
